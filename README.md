## Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/your-username/astral-home-task.git
```

2. Install dependencies:
```bash
npm install
# or
yarn install
# or
bun install
```

3. Run the project:
```bash
npm run dev
# or
yarn dev
# or
bun run dev

```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
